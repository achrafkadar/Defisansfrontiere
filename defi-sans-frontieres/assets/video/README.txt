Vidéo hero (et autres boucles courtes)
======================================

Place ici tes fichiers vidéo pour le déploiement GitHub Pages (dossier servi tel quel).

Recommandé pour le hero en arrière-plan :
- Nom suggéré : hero-loop.mp4 (tu pourras adapter le nom dans index.html / main.css une fois branché)
- Format : MP4 (H.264), piste audio retirée ou muette (l’autoplay navigateur exige souvent muted)
- Durée : boucle courte (ex. 8–20 s) pour limiter le poids du dépôt
- Résolution : 1920×1080 ou 1280×720 selon le poids cible ; compresse avant commit

Optionnel : hero-loop.webm (même contenu, souvent plus léger pour les navigateurs compatibles)

Image de repli (poster) :
- Tu peux réutiliser assets/img/hero-header-bg.png comme image affichée avant le chargement de la vidéo

Note : Git ne suit pas les dossiers vides ; ce fichier README assure que le dossier assets/video/ existe dans le dépôt. Après avoir ajouté ton .mp4, commit et push comme pour le reste du site.
